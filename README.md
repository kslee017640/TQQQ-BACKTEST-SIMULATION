# V13.2 - 묶음 비교 결과 차트 및 도움말

## 변경 내용
- 묶음 비교 결과표의 내부 스크롤을 제거하고 전체 표가 바로 보이도록 변경했습니다.
- 묶음 비교 결과 행을 클릭하면 기존 상세 팝업과 같은 형태로 차트를 표시합니다.
- 묶음 비교 상세 차트에는 종목 가격 캔들과 포트폴리오 평가금액선을 함께 표시합니다.
- 긴 설명 문구를 화면에 계속 노출하지 않고, 각 입력값 옆 `?` 도움말 버튼으로 확인하게 바꿨습니다.
- 결과표의 전략 그룹 제목에도 도움말 버튼을 추가했습니다.

## 도움말 버튼
- 레버리지 묶음, 분석 기간, 거치식 초기자본, 적립식 월 투자금, A/B/C 조건, EMA N일, 매수/매도 시그마 기준에 설명을 추가했습니다.
- 처음 사용하는 사람이 각 입력값의 의미를 팝업으로 확인할 수 있습니다.
