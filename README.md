# V14.1 - 모바일 표 스크롤 및 EMA 1주 거래

## 변경 내용
- 묶음비교 하단 결과표에 모바일 가로 스크롤을 다시 추가해 숫자가 겹치지 않게 했습니다.
- 묶음비교 상단 막대그래프의 투입금 표시를 붉은색 계열로 바꿔 더 잘 보이게 했습니다.
- EMA/시그마 전략의 매수/매도를 `보유 현금 50% 매수`, `보유 수량 25% 매도` 방식에서 `신호일마다 1주 또는 최소 $20` 방식으로 변경했습니다.
- V14.0의 환율 상단 입력, 원화 억원 보조 표기, 발동 단계별 열 분리 기능은 그대로 유지했습니다.
