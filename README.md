# V13.0 - 레버리지 묶음 비교 시뮬레이터

## 변경 내용
- 1배/2배/3배 레버리지 묶음을 비교하는 별도 기능을 추가했습니다.
- 거치식, 적립식, 하락발동 적립식 리밸런싱 3가지 방식을 같은 표에서 비교합니다.
- QQQ/QLD/TQQQ, SOXX/USD/SOXL, SPY/SSO/UPRO, TSLA/TSLL/TSLT, AAPL/AAPU/AAPB, BTC/BITU/BITX, NVDA/NVDL/NVDU 프리셋을 추가했습니다.
- 리밸런싱 발동 기준은 묶음의 1배 종목 전고점으로 통일했습니다.

## 입력값
```text
X: 거치식 초기자본 또는 적립식 월 투자금
Y: 분석 기간(년)
A: 리밸런싱 전략에서 매달 즉시 매수할 금액
B: 리밸런싱 전략에서 매달 현금으로 모아둘 금액
C: 1차 하락 발동 기준(%)
```

## 3가지 비교 방식
```text
1. 거치식 투자
   시작일에 초기자본 X를 전부 매수하고 Y년 동안 보유합니다.

2. 적립식 투자
   매달 첫 거래일에 X를 전부 매수합니다.

3. 적립식 리밸런싱 투자
   매달 첫 거래일에 A만큼 매수하고 B는 현금으로 보관합니다.
   1배 기준 종목이 전고점 대비 하락하면 보관 현금 일부로 추가 매수합니다.
```

## 5차 발동 규칙
```text
1차: -C% 하락     -> 남은 현금의 15% 사용
2차: -(C+5)% 하락 -> 남은 현금의 20% 사용
3차: -(C+10)% 하락 -> 남은 현금의 30% 사용
4차: -(C+15)% 하락 -> 남은 현금의 40% 사용
5차: -(C+20)% 하락 -> 남은 현금의 50% 사용
```

전고점을 새로 갱신하면 1~5차 발동 단계는 다시 초기화됩니다.

## 주의
- 합성 데이터가 포함된 레버리지 ETF는 실제 상장 전 구간이 추정값입니다.
- 단일종목 레버리지 ETF는 일일 리밸런싱과 변동성 끌림이 크므로 장기 결과 해석에 주의가 필요합니다.
- BTC 묶음은 BTC-USD 데이터를 기준으로 사용합니다.
