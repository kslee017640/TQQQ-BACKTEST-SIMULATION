# V14.0 - 환율 상단 입력 및 발동 열 분리

## 변경 내용
- 원화 환산 기준 환율을 묶음비교 결과표 상단에 한 번만 표시하도록 변경했습니다.
- 환율 입력칸을 추가해 사용자가 임의 환율을 직접 입력할 수 있게 했습니다.
- 현재 USD/KRW 환율을 자동으로 불러오며, 실패하면 기본값 `1달러 = 1,500원`을 사용합니다.
- 금액 칸 아래 원화 표기에서는 환율 문구를 제거하고 `(X.XX억원)`만 표시합니다.
- 묶음비교 결과표의 `발동/거래`를 `1차`, `2차`, `3차`, `4차`, `5차`, `기타 거래` 열로 분리했습니다.
