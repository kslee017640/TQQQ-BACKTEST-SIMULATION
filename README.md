# V13.9 - 묶음 기본값 및 EMA 적립식 기준

## 변경 내용
- 묶음 비교의 기본 입력값을 요청 이미지 기준으로 조정했습니다.
- 기본값: 분석기간 15년, 거치식 초기자본 35,000달러, 적립식 월 투자금 400달러, 적립식 초기 투자금 35,000달러, 리밸런싱 A 300달러, B 100달러, 하락 기준 10%, EMA 120일, 매수 1.5σ, 매도 3.0σ.
- 기준 종료일은 기존처럼 현재 날짜가 자동 입력되도록 유지했습니다.
- EMA/시그마 리밸런싱은 리밸런싱 월 정기매수 A/B가 아니라 적립식 월 투자금 X를 매달 투자하고, 별도 현금적립 없이 이전 매도 현금만 재매수에 사용하도록 변경했습니다.
